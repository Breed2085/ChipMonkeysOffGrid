# Contributing to ChipMonkeys Off Grid

## How to Help

1. **Build one.** Follow the docs, build a cell, tell us what happened.
2. **Break one.** Push it harder than we did. Find the failure modes.
3. **Share results.** Open an issue with your test data — voltage, current, runtime, what broke.
4. **Submit designs.** Got a better electrode geometry? Submit an STL via pull request.
5. **Fix docs.** Found something wrong or unclear? Edit it and submit a PR.

## Community Electrode Challenge

We test community-submitted electrode fill geometries on the YouTube channel. Design a gyroid, honeycomb, lattice, or whatever you think will work best. Submit your STL as a pull request to `electrode/community-designs/`.

Requirements:
- Must fit inside the cell cavity dimensions (see flow-battery/designs/)
- Must be printable in PLA (for sugar-soak pyrolysis)
- Include a brief description of your design rationale

## Code of Conduct

Don't be a jerk. Help each other. Share what you learn. This is open source — we're all building this together.
